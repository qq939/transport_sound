# VS Code Mac 快捷键与技巧

## 常用快捷键 (Mac)
- **Cmd + P**: 快速打开文件 (输入文件名)
- **Cmd + Shift + F**: 全局搜索
- **Cmd + Shift + O**: 跳转到符号 (函数/变量)
- **Cmd + /**: 注释/取消注释行
- **Cmd + B**: 切换侧边栏
- **Ctrl + `**: 打开/关闭终端
- **Cmd + \**: 拆分编辑器
- **Opt + Up/Down**: 上下移动当前行
- **Opt + Shift + Up/Down**: 上下复制当前行
- **Cmd + D**: 选中下一个匹配项 (多光标编辑)
- **Cmd + Shift + L**: 选中所有匹配项
- **F2**: 重命名符号

## 代码补全与智能提示
- **Tab**: 接受建议/补全代码
- **Enter**: 接受建议并换行
- **Ctrl + Space**: 强制触发代码提示 (如果没自动弹出)
- **Cmd + .**: 快速修复 (Quick Fix) / 显示操作建议

## 终端技巧
- **Cmd + K**: 清空终端
- **Cmd + T**: 新建终端标签

## Trae/AI 辅助
- 使用自然语言描述需求，AI 会帮你生成代码。
- 多使用 `Tab` 键接受 AI 的行内补全建议。
